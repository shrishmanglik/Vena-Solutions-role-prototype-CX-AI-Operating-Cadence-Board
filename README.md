# Vena CX AI Cadence Board

Role-specific prototype for the Vena Solutions Customer Experience AI Architect application.

It demonstrates a governed CX AI operating cadence:

- intake
- value, feasibility, risk, and sensitivity scoring
- RAG/source mapping
- API/tool action planning
- human approval points
- QA and evaluation checks
- release handoff and adoption playbook
- measurable impact dashboard

The prototype uses deterministic scoring and sample data only. It does not call an AI API, does not send external messages, and does not use generated image assets.

## Run

```bash
npm install
npm run dev
```

## Verify

```bash
npm run test
npm run typecheck
npm run build
```
